About

Ky is an amateur JavaScript programmer who enjoys making basic games.

These games range from simple puzzle games to 3D ones which use WebGL for GPU accelerated experiences.

Ky first got into programming when he stumbled upon Khan Academy's programming courses in late 2019, and he created his first game in November 2019.

He then started creating more of these types of projects rapidly, and made his first 3D game with WebGL in September, 2020.
